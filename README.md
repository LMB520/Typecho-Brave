# 介绍
Brave是一款拥有多功能Typecho恋爱主题，它非常适合有对象的博主使用。

# 功能
* Pjax无刷新
* 图片懒加载
* 恋爱计时器小组件
* 纪念日倒计时小组件
* 随机情话小组件
* 留言墙小组件
* 点点滴滴小组件
* 随笔说说小组件
* 关于我们小组件
* 恋爱清单小组件
* 相册小组件
* 小组件可控开关
* 适配Vaptcha人机验证
* 适配显示ip插件
* 多种可控特效
* 祝福墙违禁词
* 公告弹窗
等等等……
