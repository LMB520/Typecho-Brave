## 🎯 主题介绍

## 🚀 特色功能

### 💫 核心特性
- ✅ Pjax 无刷新加载
- ✅ 图片懒加载优化
- ✅ 组件化开关控制
- ✅ 多种可调节特效
- ✅ 适配 IP 显示插件
- ✅ 祝福墙违禁词过滤
- ✅ 公告弹窗功能

### 💖 情侣专属组件
- ❤️ 恋爱计时器
- 📅 纪念日倒计时
- 💌 随机情话展示
- 💬 留言墙功能
- 📝 点点滴滴记录
- 🗣️ 随笔说说模块
- 👫 关于我们页面
- 📋 恋爱清单管理
- 🖼️ 相册展示功能
